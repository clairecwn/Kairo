# Kairo Demo Script (~90 seconds)

## Hook (0:00–0:15)
> "You're halfway through a long integration problem. The question has scrolled off the top, the substitution you defined is three screens up, and you've lost your train of thought. Every student knows this. Kairo fixes it."

## Flow (0:15–1:15)

1. **Open the app** (question already pinned in the top band — no setup on stage). Tap it once to show the full-size peek.
2. **Write steps 1–3** of the solution with the pen. Point out: *"The ink stays exactly where I put it — no timers, no jumping. I can stop and think as long as I want."*
3. **Write steps 4–6.** As lines commit, gesture at the history gently shrinking: *"Nothing is ever hidden — older steps compact, they never disappear."*
4. **Pin the substitution** (`u = x², dx = du/2x`): tap the line → drag the handle to include both lines → Pin. The card appears on the right.
5. **Keep solving while referencing it**: hover the card — the full-size overlay opens **and stays** while writing the next step. Hover again to dismiss.
6. **Show the toolkit fast** (10 seconds): switch pen color, highlight the final answer, toggle dark mode.
7. **Finish the problem** — arms out: *"Eight steps. Question, every line, and my key reference — all on one screen. Zero scrolls."*

## Close (1:15–1:30)
> "Students stop losing their place and start finishing their reasoning. Kairo keeps the whole problem in view — the way working memory wishes notes worked."

## Safety notes
- Pre-load the demo question image and clear old ink (fresh localStorage) before going on stage.
- If Wi-Fi is down, AI recognition silently degrades — the demo flow above never depends on it.
- Walkthrough tour is skipped automatically after first run on the demo machine (or hit Skip).
